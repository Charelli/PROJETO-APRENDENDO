
GIT BASH HERE SERVE PARA DESENHAR UMA LINHA DE COMANDO EM UMA PASTA 

GIT INIT - INICIALIZAR UM NOVO REPOSITORIO 
GIT ADD COLOCAR O NOME DO ARQUIVO EM "" (nome geralmente é o que esta em vermelho) - ADICIONA OS ARQUIVOS ATUAIS AO PROXIMO CONMIT
GIT ADD . (ESPAÇO PONTO ) - ADICIONA VARIOS ARQUIVOS DE UMA VEZ 
GIT STATUS - VERIFICAR O STATUS ATUAL DOS REPOSITORIO GIT
GIT COMMIT - M 'MENSAGEM DO COMIT' - CRIA UM NOVO COMMIT COM UMA MENSAGEM 
GIT PUSH - ENVIA AS ATUALIZACOES PARA A NUVEM NA BRANCH ATUALMENTE ATIVA
GIT BRANCH - PERMITE LISTAR E VER QUAL BRANCH ESTA ATIVA ATUALMENTE
GIT CHECKOT NOME-DA-BRANCH - PERMITE MUDAR PARA UMA NOVA BRANCH 
GIR CHECKOUT-B 'NOME DA BRANCH DE ORIGEM ' 'NOME DA NOVA BRANCH' - PERMITE MUDAR E CRIAR UMA NOVA BRANCH COM BASE EM OUTRA
GIR MERGE ' BRANCH A RECEBER MERGE' - PERMITE FAZER O MERGE DA BRANCH ATIVA ATUALMENRE COM OUTRA BRANCH 
GIT PULL - ATUALIZA A BRANCH ATUALMENTE ATIVA
